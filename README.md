# RAG pipeline
This repo contains basic RAG pipeline, Q and A generator and User answer eval.

# setup
1. Make a new env
   ```bash
   python -m venv .py3
   ```
2. activate the env
   ```bash
   source ./.py3/bin/activate
   
3. install packadges from req.txt
   ```bash
   pip install -r req.py
   ```

# run
```bash
python main.py
```
